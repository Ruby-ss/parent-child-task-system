怎么测试

  1. 先在 Supabase 的 SQL Editor 执行我给你的 supabase_cloud_schema.sql 和修复 SQL。
  2. 打开本页面，填写 Project URL、anon public key、家庭码、PIN。
  3. 第一次点“创建新家庭”；第二台设备点“加入/加载家庭”。
  4. 加载成功后会自动开启同步，每 1 秒检查一次云端变化。
  5. 家长模式添加任务，孩子模式提交完成，家长模式确认加分；另一台设备会自动更新。

注意：anon public key 可以用于前端；不要把 service_role key 填进网页。
